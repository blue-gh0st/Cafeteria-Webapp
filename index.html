Hilfe

Hier sind ein paar Antworten auf häufige Fragen. Bitte erst durchlesen! Weitere Fragen bitte an info@bookfood.de

Ich habe kein Kennwort!
Wenn Du das System das erste Mal benutzen willst, so schreibe bitte eine E-Mail an info@bookfood.de

Hast Du Dein Kennwort vergessen, so benutze diesen Link link.

Denke bitte daran, dich mit deinem Namen und nicht mit deiner E-Mail-Adresse einzuloggen.

Die primäre E-Mail-Adresse auf deinem Profil muss die eines Elternteils sein. An diese E-Mail-Adressen werden alle Abrechnungen und wichtigen Informationen von bookfood.de geschickt.

Diese E-Mail-Adressen können nicht selbst geändert werden, sondern erst nach Mitteilung an info@bookfood.de.

Bestellungsbestätigungen werden an alle im Feld „Sekundäre E-Mail-Adressen“ verschickt. E-Mail-Adressen in diesem Feld können selbst geändert werden. Bitte alle E-Mail-Adressen immer aktuell halten!

Das System kennt meinen Namen nicht.
Das System kennt nur die Namen von bereits angemeldeten Benutzern.
Das System schickt mir keine automatischen E-Mails.
Du solltest binnen weniger Minuten automatische E-Mails vom System erhalten. Allerdings werde die E-Mails ggf. von Deinem E-Mail-Provider als SPAM angesehen. Ist dies der Fall, so trage bitte die E-Mail-Adresse info@bookfood.de bei Deinem E-Mail-Provider als Kontakt ein. Wenn dies nicht ausreicht, informiere uns bitte.
Der Link zur einmaligen Anmeldung funktioniert nicht!
Falls ein einfacher Klick auf den Link nicht funktioniert, kopiere bitte den gesamten Link in die Adressleiste Deines Browsers.
Bis wann kann ich bestellen?
Das Essen kann für den jeweiligen Tag bis um Mitternacht des Vortages bestellt werden.

Abbestellen ist spätestens bis zum selben Tag um 7:30 möglich.

Die Stornierung muss selbst bei bookfood.de vorgenommen werden. Melde Dich an, geh auf die Bestellliste und klicke das rote Essen an. Das Essen wird schwarz dargestellt. Speichere die Bestellung und die Stornierung ist erledigt. Andernfalls muss ich das Essen leider berechnen. Eine Stornierung per E-Mail/Telefon ist nicht möglich!

Wenn ich auf ein Essen klicke, wird es nicht rot!
Das System kann leider nicht zu 100% alle Konfigurationen unterstützen. Mögliche Probleme sind ältere Browser, kein Javascript oder Firewalls. Bitte versuche einen anderen Browser / Computer zu verwenden. Für Hinweise sind wir aber auch dankbar (Bitte Version vom Browser angeben).
Ich habe aus Versehen mehrmals bestellt. Kann ich eine Bestellung stornieren?
Es wird pro Kind und Woche nur eine Bestellung im System gespeichert. Die letzte Bestellung zählt dabei, sprich die vorherigen werden überschrieben (siehe Startseite Bestellen).
Ich habe falsch bestellt. Was kann ich tun?
Bestellungen können bis Mitternacht des Vortages geändert werden (siehe Startseite Bestellen).
Was passiert mit meinen Bestellungen, wenn ich krank bin.
Wenn Du krank bist, musst Du das Essen noch bis 7:30 des selben Tages abbestellen. Melde Dich an, geh auf die Bestellliste und klicke das rote Essen an. Das Essen wird schwarz dargestellt. Speichere die Bestellung und die Sache ist erledigt. Andernfalls muss ich das Essen leider berechnen. Eine Stornierung per E-Mail/Telefon ist nicht möglich.
Wie erfahre ich meinen Kontostand?
Der Kontostand steht im Profil. Deine Eltern erhalten aber auch jeden Tag, an dem Du etwas bestellt hast, eine Abrechnungs-E-Mail für diesen Tag und das aktuelle Guthaben.
Können meine Eltern auch eine E-Mail über meine Bestellungen erhalten?
Ja, im Profil kann man bei Sekundäre-E-Mail auch mehrere durch Komma getrennte Adressen (ohne Leerzeichen, Semikolon etc.) eingeben. So sind Deine Eltern über Deine Bestellungen informiert.

Warum hat das System keine Benutzernamen?
Dies ist ein System in dem jeder Benutzer mit voller Verantwortung unter seinem richtigen Namen arbeitet. Moderne Browser unterstützten das Vorausfüllen von Feldern.
Einige Informationen scheinen von einem anderen Schüler zu sein!
Bitte benutze das System nur in einem Browserfenster - also nie gleichzeitig in mehreren - und melde Dich nach der Benutzung immer ab.
Das Profil ist falsch. Was kann ich tun?
Das Profil enthält Informationen, die Du mit Thomas Beier bzw. bookfood.de teilst. Du kannst und solltest diese auch selber ändern.Achte beim Ändern deiner Profildaten darauf Änderungen immer abzuspeichern.
Ich habe einen Berlinpass. Was muss ich tun?
Registriere Dich zuerst und melde Dich dann bitte bei Thomas. Er wird Dein Konto freischalten und Du kannst anfangen zu bestellen (Nur genau ein Essen pro Tag).
Ich bekomme englische Texte / Meldungen.
Das System hat ggf. noch "Überreste" von einer englischen Version. Bitte teile uns die Texte mit.
Ich brauche noch immer Hilfe!
